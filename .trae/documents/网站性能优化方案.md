## 网站性能优化计划

### 1. 字体优化（核心优化）
- 将 TTF 字体转换为 WOFF2 格式（减少 30-50% 大小）
- 使用字体子集化工具提取常用字符（减少到 1-2MB）
- 修改 `font-display` 为 `fallback` 以优化加载体验

### 2. MathJax 延迟加载
- 修改 MathJax 配置，实现按需加载
- 只在包含数学公式的页面加载 MathJax

### 3. CSS 优化
- 检查并合并可合并的 CSS 文件
- 移除未使用的样式

### 4. 配置优化
- 添加字体预加载提示
- 优化资源加载顺序

**预期效果**：首屏加载时间从 5-10 秒优化到 1-2 秒
