# 线性代数核心内容总结（第1-2章）

## 一、线性方程组的求解理论

### 1. 线性方程组的基本概念
- 形式：\( a_{11}x_1 + a_{12}x_2 + \dots + a_{1n}x_n = b_1 \)（共 \(m\) 个方程）
- 未知量：\( x_1, x_2, \dots, x_n \)
- 系数：\( a_{ij} \)（第 \(i\) 个方程中 \(x_j\) 的系数）
- 常数项：\( b_i \)
- 齐次/非齐次：常数项全为零则为齐次
- 解：一组数 \( c_1, c_2, \dots, c_n \) 代入方程组所有方程均成立

### 2. 线性方程组的初等变换
- **互换**：交换两个方程的位置
- **倍乘**：用非零常数乘某个方程
- **倍加**：将一个方程的常数倍加到另一个方程上
- 初等变换保持方程组的**同解性**

### 3. 阶梯形线性方程组
- 通过初等变换可将方程组化为**阶梯形**（形如定理1中的(1.2.1)）
- 阶梯头：每个阶梯中第一个非零系数的项
- 阶梯形方程组便于判断解的存在性与求解

### 4. Gauss消元法
- 利用初等变换将方程组化为阶梯形
- 判断解的条件：
  - 有解：\( c_{r+1} = 0 \) 或 \( r = m \)
  - 无解：\( c_{r+1} \neq 0 \)
  - 唯一解：\( r = n \)
  - 无穷多解：\( r < n \)
- 通解表示：用自由未知量（自由变量）表示所有解

## 二、矩阵的引入与初等变换

### 1. 矩阵的定义
- \( m \times n \) 矩阵：由 \( m \times n \) 个数按矩形排列的数表
- 表示：\( A = (a_{ij})_{m \times n} \)
- 特殊矩阵：
  - 零矩阵：所有元素为零
  - 方阵：\( m = n \)
  - 分块矩阵：将矩阵划分为若干子块

### 2. 矩阵的初等变换
- **初等行变换**：
  - 互换：\( R_{ij} \)（交换第 \(i, j\) 行）
  - 倍乘：\( cR_i \)（第 \(i\) 行乘非零常数 \(c\)）
  - 倍加：\( R_i + cR_j \)（第 \(j\) 行的 \(c\) 倍加到第 \(i\) 行）
- 初等列变换类似（记作 \( C_{ij}, cC_i, C_i + cC_j \)）
- 初等变换是Gauss消元法的矩阵表达形式

### 3. 增广矩阵与线性方程组
- 系数矩阵：\( A = (a_{ij})_{m \times n} \)
- 增广矩阵：\( \overline{A} = (A | b) \)（包含常数项）
- 初等行变换不改变线性方程组的解

## 三、行列式

### 1. 排列与逆序数
- \(n\)-排列：由 \(1,2,\dots,n\) 组成的一个有序序列
- 逆序数：\( \tau(i_1 i_2 \dots i_n) \) 表示排列中逆序对的个数
- 奇排列/偶排列：逆序数为奇数/偶数
- 对换改变排列的奇偶性

### 2. 行列式的定义
- \( n \) 阶行列式定义：
\[
|A| = \sum_{j_1 j_2 \dots j_n} (-1)^{\tau(j_1 j_2 \dots j_n)} a_{1j_1} a_{2j_2} \dots a_{nj_n}
\]
- 其中求和遍历所有 \(n\)-排列

### 3. 行列式的性质
- 转置不改变行列式值：\( |A| = |A^T| \)
- 交换两行（列）：行列式变号
- 某行（列）乘常数 \(c\)：行列式乘 \(c\)
- 两行（列）成比例：行列式为0
- 某行（列）可拆分为两行（列）之和：行列式可拆分
- 倍加变换不改变行列式值

### 4. 行列式的计算
- 三角形行列式：值为主对角线元素的乘积
- Vandermonde行列式：
\[
D_n = \prod_{1 \leq i < j \leq n} (x_j - x_i)
\]
- 按行（列）展开（Laplace定理）
- 代数余子式：\( A_{ij} = (-1)^{i+j} M_{ij} \)

### 5. Cramer法则
- 若系数行列式 \( D \neq 0 \)，则方程组有唯一解：
\[
x_j = \frac{D_j}{D}, \quad j=1,2,\dots,n
\]
- 其中 \( D_j \) 是将 \(D\) 中第 \(j\) 列换为常数项所得行列式

## 四、矩阵的秩

### 1. 秩的定义
- 矩阵中**非零子式的最高阶数**称为秩
- 记作 \( r(A) \)
- 零矩阵的秩为0
- 性质：\( 0 \leq r(A) \leq \min\{m, n\} \)

### 2. 秩的等价定义
- \( r(A) \geq r \)：存在一个非零的 \(r\) 阶子式
- \( r(A) \leq r \)：所有 \(r+1\) 阶子式（若有）全为零

### 3. 秩的性质
- 矩阵的秩是**初等变换的不变量**
- 矩阵增加一行（列），秩至多增加1
- 可通过初等变换将矩阵化为阶梯形，其非零行数即为秩

---

## 五、核心关联
- **线性方程组** → **矩阵表示** → **初等变换** → **阶梯形** → **解的结构**
- **行列式** 用于判断方程组解的唯一性（Cramer法则）
- **矩阵的秩** 是解的存在性与个数的关键指标
- **初等变换** 是连接方程组求解、矩阵化简、行列式计算、秩计算的统一工具
